MVC Traditional Architecture

User navigates to home page; server returns the home page.
User navigates to another page; server generates and returns another HTML page.
Controllers within the web server acesss the model to generate HTML pages.

MVC Single Page Application Architecture

User navigates to home page; server returns an entire application.
Application displays home view.
Views request view models (json) from the web server.

MVC ASP.NET Core 2.1 SPA Architecture

User navigates to home page; Kestrel server constructs home controller.
Home controller returns entire application.
Controllers are constructed as needed to handle requests from the browser.

The SilverLight client maps well to Angular / ASP.NET Core MVC
