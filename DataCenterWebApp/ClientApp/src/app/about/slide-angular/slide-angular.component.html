Architecture Overview

NgModules

Collection of related Components, Services, Directives and Pipes

Components define views

Metadata specifies a "template" (HTML with directives)
Data binding between the HTML and component class

Services provide the functionality

Injected into the components when constructed
Built in services include http, router, timer

Directives

Structural - manipulate the elements within the DOM
Attribute - change the appearance or behavior of a DOM Element

Pipes

Takes in data and transforms it as output
Built in pipes include UpperCasePipe, DatePipe
Custom pipes
experimentfilter.pipe => experiments are filtered by search criteria
highlighter.pipe => string is modified to "highlight" substrings
